Returns:

Examples

Parse tab separated data into a table:

>>> from biom.table import Table
>>> from io import StringIO
>>> tsv = 'a\tb\tc\n1\t2\t3\n4\t5\t6'
>>> tsv_fh = StringIO(tsv)
>>> func = lambda x : x
>>> test_table = Table.from_tsv(tsv_fh, None, None, func)
